Media Library Dashboard

A self-hosted media library dashboard for Radarr, Sonarr, and Jellyseer. SQLite-backed, Docker-ready, zero cloud dependency.

View on GitHub Releases Issues
Features
🎬
Movies & TV Stats
KPIs, codec breakdown, resolution, HDR type, total size, upgrade queue — all from your Radarr and Sonarr libraries.
🧬
Library DNA
Multi-dimensional scoring and letter-grade system. Measures score authenticity, intentionality, talent crossover, franchise context, and more.
📦
Bloat Analysis
Surface oversized encodes and trigger one-click Radarr replacement — no manual file hunting required.
🎯
Franchise Tracker
Collection completion with drill-down to missing titles. See which franchises are complete vs. in-progress.
🌟
Talent Deep Dive
Everywheremen, Unsung Heroes, BTWF, Deep Wounds — powered by IMDb data matched to your library.
📈
Trend History
Sparklines and history table across all pipeline runs. Browse any past snapshot via the run dropdown.
🧪
Family Fingerprint
Per-member taste profiles, genre overlap, pairwise similarity, and personalized recommendations from your library and IMDb.
OPTIONAL
😤
Hated Titles
Track titles tagged for removal per member. Surfaces a shared expulsion list of D/F-graded hated titles.
OPTIONAL
Requirements
Radarr v3+
Sonarr v3+
Jellyseer / Overseerr
Docker + Compose
Python 3.10+
Quick Install

Run the interactive setup — it validates your API connections and writes config.py automatically.

$ git clone https://github.com/fsimpson-code/media-library-dashboard
$ cd media-library-dashboard
$ python3 setup.py
$ docker compose up -d
